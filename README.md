# University Timetable Scheduling Using Adaptive-Elitist Genetic Algorithm


# Installation
first of all install all the requirements using
    
    pip install -r requirements.txt 

### Usage:
1. Add instructors
2. Add rooms
3. Add subjects
4. Add sections
5. Generate solution
6. Export or View Result

### Dependencies:
1. Numpy
2. PyQT5
3. psutil
4. logging
5. xlsxwriter
